🧬 RNA-seq Differential Expression Analysis (Cancer vs Normal)
📌 Project Overview

This project performs RNA sequencing (RNA-seq) analysis to identify differentially expressed genes between cancer and normal samples using DESeq2.

🧪 Dataset

Data was obtained from public repository of National Center for Biotechnology Information GEO database.

⚙️ Methods
Data preprocessing and normalization
Differential expression analysis using DESeq2
Principal Component Analysis (PCA)
Heatmap visualization
Volcano plot generation
📊 Key Outputs
PCA plot showing sample clustering
Heatmap of top expressed genes
Volcano plot of gene significance
Differential expression results table
🧰 Tools Used
R
DESeq2
GEOquery
ggplot2
pheatmap
📁 Results

All plots and results are stored in the results/ folder.

🚀 Skills Demonstrated
RNA-seq data analysis
Statistical modeling (negative binomial)
Data visualization
Bioinformatics pipeline development
